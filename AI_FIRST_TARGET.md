APP_IDENTITY:
Padely est un réseau social compétitif pour joueurs de padel.

CORE_LOOP:
Joueur joue → enregistre match → gagne/perd points → consulte classement → veut revanche

WHAT MAKES THE APP GOOD:
- friction minimale pour enregistrer un match
- plaisir de voir sa progression
- rivalités visibles
- sentiment de niveau réel

PRIORITIES_ORDER:
1 User experience
2 Engagement / addiction
3 Features
4 Code quality

EVERY CHANGE MUST:
- réduire le nombre de taps
- rendre un écran plus clair
- ajouter motivation de rejouer
- ou ajouter information utile joueur

BAD CHANGES:
- refactor inutile
- architecture parfaite sans impact utilisateur
- complexité interne invisible

GOOD CHANGES:
- nouveau feedback utilisateur
- meilleure lisibilité
- automatisation
- suggestions intelligentes
