# Padely - Plan Produit

## Positionnement

Application padel "all-in-one" pour les joueurs en France:

- suivi performance (PIR + stats)
- social et matchmaking
- coaching intelligent via donnees montre

## MVP (Phase 1)

### Objectif
Livrer un premier produit utilisable en club avec un flux simple:

1. creer un profil
2. lancer un match
3. saisir/valider le score
4. mettre a jour PIR
5. visualiser progression

### User stories prioritaires

- En tant que joueur, je veux enregistrer un match pour mettre a jour mon niveau.
- En tant que joueur, je veux valider un score recu pour eviter les abus.
- En tant que joueur, je veux voir mon historique PIR dans le temps.
- En tant que joueur, je veux trouver des joueurs de niveau proche.

## Roadmap

### Phase 1 (MVP)

- app mobile scoring simple
- profil utilisateur
- PIR de base

### Phase 2 (Connectivite)

- Apple HealthKit et Google Fit
- app montre pour scoring rapide

### Phase 3 (Intelligence)

- analyse IA des baisses de performance
- recommandations de drills personnalisees
- beta PIR puissance smash

### Phase 4 (Social)

- saisons par ville
- badges et recompenses
- marketplace locale

## KPI de lancement

- MAU (joueurs actifs mensuels)
- taux de matchs valides (cross-validation)
- retention S4
- pourcentage de matchs "equilibres" (prediction 40-60)
