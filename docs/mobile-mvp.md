# Mobile MVP - Ecrans et navigation

## Arborescence simple

1. Home
2. Jouer
3. Communaute
4. Stats/Materiel

## Ecran Home

- prochain match
- score PIR actuel
- badge en cours
- bouton "Lancer un match"

## Ecran Jouer

- creation match 2v2
- scoring gros boutons (15/30/40/jeu/set)
- option Punto de Oro
- validation croisee post-match

## Ecran Communaute

- recherche joueur par PIR
- filtres distance et disponibilite
- clubs proches sur carte

## Ecran Stats/Materiel

- courbe PIR
- radar PIR DNA
- stats binome
- sac de padel + usure raquette

## Contraintes UX

- mode sombre obligatoire
- interactions a gros targets (smartphone + montre)
- actions critiques en 1-2 taps max
